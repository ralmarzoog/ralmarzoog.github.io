  • EcoSym

    Collaborated with Nathan Stocking and Ben deButts

    Simulated events in a marine ecosystem with concurrently running threads using Python

    EcoSym is a simulation of a marine ecosystem I and my teammates created for our Concurrent Programming class. The ecosystem contains marine organisms and simulates the relationships between these organisms, as well as a few natural processes that occur in such ecosystems. EcoSym currently simulates a select few marine organisms, but the design allows potential to quickly add more. The size of the body of water and number of organisms is only limited by the thread limit on the machine running the simulation. Features include predator-prey relationships, reproduction, aging and maturity, movement, and starvation.

    The project primarily uses object polymorphism and separate threads to represent organisms or groups thereof. There is a central ecosystem class which contains the shared state of the ocean and stores data and functions allowing organisms to read and affect changes to the ocean.
